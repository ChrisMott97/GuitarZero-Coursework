README

To compile from current directory:
1) download javaFX 11 from: https://gluonhq.com/products/javafx/
2) compile with command: javac --module-path [/path/to/javafxSDK/lib] --add-modules=javafx.controls Play.java
3) run with command: java --module-path [/path/to/javafxSDK/lib] --add-modules=javafx.controls Play
